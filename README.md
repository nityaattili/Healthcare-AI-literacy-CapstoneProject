# Healthcare-AI-literacy-CapstoneProject
Capstone project on AI literacy in healthcare using NLP and knowledge mapping

## Overview
This project focuses on AI literacy in healthcare data. It combines literature analysis using NLP techniques with the design of a conceptual web-based platform for healthcare AI literacy.

## Project Goals
1.Collect healthcare-related AI literacy literature
2.Clean and analyze text data from research papers
3.Identify themes, concepts, and patterns using NLP
4.Design an AI literacy platform for healthcare users

## Project Structure
- `data/` → raw, processed, and external datasets
- `docs/` → proposal, methodology, and reports
- `notebooks/` → Jupyter notebooks for analysis
- `src/` → Python scripts for data collection, preprocessing, and NLP
- `results/` → figures, tables, and outputs
